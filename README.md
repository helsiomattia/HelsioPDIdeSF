# Mapa de Competências Salesforce

## Visão Geral

Bem-vindo ao meu portfólio online!  
Eu sou **Helsio Mattia**, um Certified Salesforce Professional com sólida experiência em otimização de CRM, análise de requisitos, automação de processos e liderança de times. Este projeto foi desenvolvido para demonstrar minhas competências e experiências em diversas áreas do ecossistema Salesforce, incluindo a administração, consultoria e arquitetura de soluções.

Utilizando uma interface moderna e responsiva, o site apresenta:
- **Páginas específicas** para diferentes papéis: Administrador, Consultor e Architect.
- Um **menu dropdown dinâmico** (carregado via um arquivo compartilhado `header.html`) para facilitar a navegação.
- **Seções bem estruturadas** que exibem meu perfil, experiências profissionais, formação, certificações, competências e idiomas.
- Um design que integra cores e elementos modernos (baseado em W3.CSS, Google Fonts e Font Awesome) para garantir uma aparência corporativa e atrativa.

## Funcionalidades

- **Página Inicial (Index):**  
  Apresenta meu perfil profissional com foto, resumo, experiências, formação, certificações, skills e contato.

- **Menu Dropdown Dinâmico:**  
  O menu é centralizado em um arquivo único (`header.html`), que é carregado dinamicamente em todas as páginas para manter consistência na navegação.  
  - **Admin:** Links para Hard Skills e Soft Skills.
  - **Consultor:** Links para Hard Skills e Soft Skills.
  - **Architect:** Links para Hard Skills e Soft Skills.

- **Armazenamento Local:**  
  Notas e pontuações das competências são armazenadas localmente (via Local Storage), permitindo que as seleções do usuário sejam mantidas entre as sessões.

- **Estrutura Modular para Futuras Funcionalidades:**  
  O projeto está organizado em seções e pastas separadas, facilitando a manutenção e a futura integração de componentes como gráficos de evolução das pontuações.

## Tecnologias Utilizadas

- **HTML5** e **CSS3** para estrutura e estilo.
- **JavaScript** para lógica dinâmica (fetch do header, armazenamento local, etc.).
- **W3.CSS** para um layout moderno e responsivo.
- **Google Fonts (Roboto)** para uma tipografia limpa e profissional.
- **Font Awesome** para ícones.
- **Local Storage** para persistência dos dados do formulário.
